# 구조

- 던전형 싱글 로그라이크 카드게임



## 페이지

1) 메인 페이지 (각 버튼: 게임시작 / 덱 구성 / 설정)

2) 게임 보드 페이지

[][][][][]      적 몬스터 카드 존(5칸) 
    []          공용 묘지(1칸) 
[][][][][]      플레이어 몬스터 존(5칸)
    []          플레이어 덱(1칸) 
[][][][][]      플레이어 카드 패(최소 0칸 ~ 최대 5칸)   
  []  []        플레이어 체력(1칸) - 금화(1칸)

3) 상점 페이지

- 전투 종료시, 상점에서 금화로 카드를 구매하거나 플레이어 체력, 몬스터 체력을 회복할 수 있다.




## 게임 진행

- 플레이어는 게임 시작 전, 덱을 구성한다. (최대 20장)

[카드종류]
- 몬스터 : 코스트는 금화 1 ~ 5개, 코스트가 많이 들수록 강력한 몬스터
- 스펠: 코스트는 금화 1 ~ 5개, 코스트가 많이 들수록 강력한 스펠카드

### 인게임

[기본규칙]
- 플레이어는 기본적으로 4000의 체력, 금화 5개를 가진다.
- 게임 시작시, 덱에서 카드를 5장 뽑는다.
- 플레이어는 랜덤으로 생성되는 몬스터 카드를 마주친다.
- 매 턴, 덱에서 카드를 1장 뽑는다. 패의 최대 매수는 5장이고 패가 5장일 경우 카드를 뽑을 수 없다.
- 플레이어가 가지고 있는 금화에 따라, 패에서 몬스터나 스펠 카드를 필드위에 꺼낼 수 있다.
- 적 몬스터를 물리치면 금화를 얻을 수 있다. 플레이어가 얻을 수 있는 금화는 상대 몬스터의 코스트 금화이다.
- 총 라운드는 50회
- 한 번 소환된 플레이어의 몬스터는 계속 필드에 남아있고, 체력도 전투가 끝난 상태 그대로다. 따라서 몬스터를 따로 회복하지 않으면 회복되지않는다.


[전투]
- 플레이어는 전투에서 데미지를 받고, 체력이 0이 되면 게임 오버.
- 상대 몬스터를 처치하면 다음 라운드로 넘어간다.
- 상대 몬스터 카드는 여러 장일 수 있다.
- 처치된 적, 플레이어 몬스터 카드는 묘지로 간다. 묘지로 간 몬스터는 특별한 수단(부활 카드 등)이 없으면 부활이 불가능하다.
