# 구조

- 던전형 싱글 로그라이크 카드게임

## 카드 종류

- 몬스터
- 스펠

## 게임 보드 구성

- 몬스터 카드(적) - 묘지 - 플레이어 몬스터 존 - 플레이어 카드 패 - 플레이어 덱 - 플레이어 체력 - 게임 머니

## 게임 진행

- 플레이어는 게임 시작 전, 덱을 구성한다. (최대 30장)

### 인게임

[기본규칙]
- 플레이어는 4000의 체력을 가진다.
- 게임 시작시, 덱에서 카드를 5장 뽑는다.
- 플레이어는 랜덤으로 생성되는 몬스터 카드를 마주친다.
- 매 턴, 덱에서 카드를 1장 뽑는다.
- 적 몬스터를 물리치면 게임 머니를 얻을 수 있다. 
- 총 라운드는 50회
- 한 번 소환된 플레이어의 몬스터는 계속 필드에 남아있고, 체력도 전투가 끝난 상태 그대로다. 따라서 따로 회복하지 않으면 회복되지않는다.
- 전투 종료시, 상점에서 게임 머니로 카드를 구매하거나 플레이어 체력, 몬스터 체력을 회복할 수 있다.
- 플레이어의 차례가 끝날 시 가지고 있는 카드는 6장 이하여야 한다. 6장이 넘어가면 6장이 될 때까지 카드를 묘지에 버려야한다.

[전투]
- 플레이어는 전투에서 데미지를 받고, 체력이 0이 되면 게임 오버.
- 상대 몬스터를 처치하면 다음 라운드로 넘어간다.
- 상대 몬스터 카드는 여러 장일 수 있다.
- 처치된 적, 플레이어 몬스터 카드는 묘지로 간다. 묘지로 간 몬스터는 특별한 수단(부활 카드 등)이 없으면 부활이 불가능하다.
